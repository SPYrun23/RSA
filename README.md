# RSA
# 📊 Retail Sales Analysis Dashboard

## 📌 Overview
This project analyzes retail sales data and provides insights using visualization and forecasting.

## 🚀 Features
- Data preprocessing
- KPI analysis
- Interactive dashboard (Streamlit)
- ARIMA-based sales prediction
- Download processed data

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Streamlit
- ARIMA (Statsmodels)

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
